# Unsolicited Podcast

A premium, responsive podcast landing page optimized for high-performance hosting on platforms like Vercel or Netlify.

- **Dynamic "Focus" Hero:** Automatically highlights the selected episode with a full description and topic-relevant album art (via Unsplash API).
- **Glassmorphism Persistent Player:** A sleek, sticky bottom player that ensures uninterrupted listening while browsing the catalog.
- **Optimized Audio Engine:** All episodes are converted to high-quality 128kbps MP3s, optimized for instant, buffer-free streaming.
- **"Quiet" Active State:** Minimalist UI that focuses attention on the active content by hiding redundant controls on playing cards.
- **Refined Typography:** Premium editorial aesthetic using 'Inter' for bold headers and 'Merriweather' for readable serif descriptions.
- **Accessibility & UX:** Includes dark/light mode, keyboard shortcuts (Spacebar to play/pause), and a "Jump to Top" feature.
